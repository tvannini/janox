Nuovo Janox Mini WEB Server
14.06.2017 | Janox news

La versione in sviluppo 2.3.00 introduce un nuovo potente strumento: Janox Mini WEB Server.

Se volete provare l'applicazione Demo, eseguire dei test o eseguire in maniera facile e veloce applicazioni Janox da oggi sarà possibile farlo senza aver bisogno di un WEB Server configurato: basterà avere un eseguibile PHP 5.4 o superiore.

Per una documentazione dettagliata del nuovo strumento visita il nostro Wiki Janox Mini WEB Server.

Consolidata la nuova versione Janox 2.2.01
05.03.2017 | Janox news

La Community Janox è lieta di annunciare il consolidamento della versione 2.2.01.

Fra le principali novità di questa release ricordiamo:

  • la possibilità di spostare l'ordine delle colonne in una grid;
  • l'utilizo di statement persistenti per i le action in loop su una vista (solo Postgres);
  • autenticazione intranet tramite LDAP.

La versione 2.2.01 è stabile ed ampiamente testata ed è consigliata anche per installazioni di produzione.
