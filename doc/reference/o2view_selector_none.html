o2view_selector_none

boolean o2view_selector_none(string $view_name, boolean $incremental)

Unselect all selected rows of a view in a view-selector structure.
If parameter $incremental is passed as TRUE, only rows currently filtered in the view will be unselected, otherwise all selected rows will be unselected.


Parameters:
  1. $view_name
  2. $incremental
