(misc)

o2view_export

string o2view_export(string $view_name, string $csv_file_name, boolean $only_visible, string $fields_separator, string $text_delimiter, string $decimal_separator, string $date_format, string $time_format, boolean $titles_from_view, array $excluded_aliases, array $used_aliases)

Export data from a prg view to a CSV format file.
File is zipped and the full path to file is returned.


Parameters:
  1. $view_name
    2. Name of the view to export data from
  2. $csv_file_name
    3. Target file name
  3. $only_visible
    4. Export only data visible on prg forms
  4. $fields_separator
    5. Character use to separate fields
  5. $text_delimiter
    6. Character used to delimiter text values
  6. $decimal_separator
    7. Character used as decimal point in numbers
  7. $date_format
    8. Format used for data fields (PHP date() style)
  8. $time_format
    9. Format used for time fields (PHP date() style)
  9. $titles_from_view
    10. If TRUE field alias in view is used as title
  10. $excluded_aliases
    11. List of aliases excluded in export
  11. $used_aliases
    12. List of (alias => column-title) to override all
