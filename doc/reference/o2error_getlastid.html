(error)

o2error_getlastid

integer o2error_getlastid(boolean $box_div)

Returns last error unique id.
If parameter $box_div is passed as TRUE, then numeric id is prefixed with "jxerror_": this parameter is useful to get the message box div id in custom error files.


Parameters:
  1. $box_div
    2. Numeric unique id or box div id
