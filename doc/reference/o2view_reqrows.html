o2view_reqrows

integer o2view_reqrows(string $view_name)

Returns number or requested records for actual paging.
Requested records number is set by the view property "Requested rows number".
Note: requested records number can differ from actually returned records number, when recordset end is reached.


Parameters:
  1. $view_name
