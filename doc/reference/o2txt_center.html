o2txt_center

string o2txt_center(string $text, integer $width, integer $indent, string $indent_chr, string $fill_chr)

Format text as splitted in two parts, one aligned to left, the other aligned to right


Parameters:
  1. $text
    2. Text to be aligned to left
  2. $width
    3. Width to format text to
  3. $indent
    4. Number of characters for indentation
  4. $indent_chr
    5. Character to be used for indentation
  5. $fill_chr
    6. Character to be used to fill space out of text
