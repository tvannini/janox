o2job_is_scheduled

boolean o2job_is_scheduled(string $job_name)

Returns TRUE if job with passed name is scheduled, else FALSE.
NOTE: if session ASPID is set then only scheduled items for current ASPID are considered (ASP blank, 'NOASP' or equal to current ASPID).


Parameters:
  1. $job_name
    2. Name to look for
