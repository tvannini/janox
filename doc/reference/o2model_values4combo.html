(model)

o2model_values4combo

array o2model_values4combo(string $model_name, boolean $add_blank, string $blank_label)

Returns an array of model possible values as defined in picture.
If second parameter $add_blank is passed as TRUE, then blank ('' or 0) value is added to list, where not already present.
This is mostly useful for feeding combos filtering $model_name related data.


Parameters:
  1. $model_name
  2. $add_blank
  3. $blank_label
