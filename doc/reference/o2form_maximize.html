(form)

o2form_maximize

boolean o2form_maximize(string $form_name, boolean $maximized)

Maximize or restore form size.
When maximized the form takes all the available space on client browser.
When $maximized parameter is passad as TRUE (default) form is maximized, if not already maximized.
When parameter $maximized is passed as FALSE, if form is maximized, then it is reset to its original size and position.


Parameters:
  1. $form_name
  2. $maximized
