o2_date_diff

integer o2_date_diff(string $date1, string $time1, string $date2, string $time2, string $format)

Returns the difference between two dates/times, passed in Janox format.
If last parameter $format is not passed, then difference is returned in number of days.
If last parameter $format is passed, then it is used as PHP date/time format string.


Parameters:
  1. $date1
    2. Start date in Janox format
  2. $time1
    3. Start time in Janox format
  3. $date2
    4. End date in Janox format
  4. $time2
    5. End time in Janox format
  5. $format
    6. PHP date/time format string
