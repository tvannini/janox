o2job_create

integer o2job_create(string $name, string $prg, array $params, string $desc, integer $keep_days, integer $max_instances, string $block_mode, boolean $disabled)

Creates a new job, assigns a new ID and stores it in system data.
New assigned ID is returned.


Parameters:
  1. $name
    2. Job name
  2. $prg
    3. Program name caled by job
  3. $params
    4. List of program parameters, referenced by name
  4. $desc
    5. Job description
  5. $keep_days
    6. Days job instances are preserved from deleting
  6. $max_instances
    7. Max number of instances can run at a time
  7. $block_mode
    8. Run blocking mode: [B]lock, [Q]ueue
  8. $disabled
    9. Disabled job will not run
