o2job_proc_restart

string o2job_proc_restart(integer $runjob_id, boolean $preserve_id)

Starts over an existing process.
If parameter $preserve_id is passed TRUE the same process is set to queued and reused, else, if FALSE, a new process is created with the same parameters and APSID of the original one.


Parameters:
  1. $runjob_id
    2. Run-job (process) ID to look for
  2. $preserve_id
    3. If the same process is to be reused
