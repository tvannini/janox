(table)

o2tab_trace

string o2tab_trace(string $table_name)

If record tracing is enabled for table returns a string with record tracing settings, else FALSE.
Possible tracing settings are:
- cu - creation user
- cd - creation date
- ct - creation time
- uu - update user
- ud - update date
- ut - update time


Parameters:
  1. $table_name
