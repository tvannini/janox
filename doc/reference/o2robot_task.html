(robot)

o2robot_task

o2robot_task(string $id, integer $freq, string $prg_name)

Adds a task to be executed on a certain frequency: each time task is executed a time marker is reset to evaluate next frequency expiration.


Parameters:
  1. $id
    2. Unique name for the task
  2. $freq
    3. Execution frequency in hours
  3. $prg_name
    4. Program name to execute for task
