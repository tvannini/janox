string o2app_alias()

Returns application alias as an URL prefix, mostly useful to address application resources (images, JS libraries, ...)

NOTE: application alias is returned with ending back-slash.

Sample: if you are serving your application from "[myhost]/apps/myapp/myapp.php" main page, then application alias will be returned as "/apps/myapp/".
