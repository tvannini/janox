integer o2dispatch_add(string $title, string $body, string $icon)

Adds a new dispatch to dispatcher list and return assigned unique ID.


Parameters:
  1. $title
    2. Title, visible in the list, to assign to dispatch
  2. $body
    3. Text of the dispatch, visible in the tooltip
  3. $icon
    4. Icon to show for the dispatch in the list
