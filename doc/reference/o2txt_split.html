o2txt_split

string o2txt_split(string $text_left, string $text_right, integer $width, integer $indent, string $indent_chr, string $sep_chr)

Format text as splitted in two parts, one aligned to left, the other aligned to right


Parameters:
  1. $text_left
    2. Text to be aligned to left
  2. $text_right
    3. Text to be aligned to right
  3. $width
    4. Width to format text to
  4. $indent
    5. Number of characters for indentation
  5. $indent_chr
    6. Character to be used for indentation
  6. $sep_chr
    7. Character to be used to fill space between textes
