o2tree_export

string o2tree_export(string $tree_name, string $csv_file_name, string $fields_separator, string $text_delimiter)

Export data from a TreeView control to a CSV format file.
File is zipped and the full path to file is returned.


Parameters:
  1. $tree_name
    2. Name of the TreeView control to export data from
  2. $csv_file_name
    3. Target file name
  3. $fields_separator
    4. Character use to separate fields
  4. $text_delimiter
    5. Character used to delimiter text values
