o2dispatch_set_prg

boolean o2dispatch_set_prg(integer $id, string $prg_name)

Set the activation program for the requested ID.
Returns TRUE on success and FALSE on fail.
To pass parameters for activation program just pass them after $prg_name.


Parameters:
  1. $id
    2. Dispatch ID to set program for
  2. $prg_name
    3. Activation program to call on click on dispatch
